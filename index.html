Fluxo — IA Sourcing (do input do cliente ao resultado)
01
Input do Cliente
Cliente digita necessidade em linguagem natural.

"Air fryer 12L, acabamento cobre fosco, vender a R$ 800"
02
Processamento RAG
Texto vira embedding vetorial. Busca por similaridade no Pinecone com catálogo MU Group + API 1688.
03
Cálculo DDP
API Fiscal retorna: II + IPI + ICMS + PIS/COFINS pelo NCM. Frete FOB → DDP calculado em tempo real.
04
Output Estruturado
3 fabricantes + preço DDP + margem sugerida + botão "Solicitar Amostra" ou "Add ao Container".
Fluxo — Rastreamento (da fábrica na China ao cliente no Brasil)
01
Fábrica → Porto CN
Equipe China atualiza status via Painel. Fotos e vídeos da inspeção enviados ao dashboard do cliente.
02
Trânsito Marítimo
AIS API rastreia posição do navio por IMO number. Webhook atualiza o portal a cada 4h automaticamente.
03
Porto BR → Despacho
Intelipost + API SEFAZ notifica chegada e status do desembaraço aduaneiro em tempo real.
04
Última Milha
Transportadora integrada via Intelipost. Cliente recebe push e email em cada etapa até a entrega.
